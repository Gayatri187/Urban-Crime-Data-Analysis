# Urban Crime Data Analysis

## Overview
This project analyzes urban crime data to identify patterns, trends, hotspots, and time-based crime behavior using Python and data visualization techniques.

The analysis explores crime types, yearly and monthly trends, high-crime districts, time-of-day patterns, and geographic crime density using mapping tools.

## Objectives
The main goals of this project are:

- Analyze the most common crime types
- Study crime trends over time
- Identify districts with high crime rates
- Understand crime distribution by month and hour
- Visualize geographic crime hotspots using maps
  
## Tools and Technologies
This project was developed using:

- Python
- Pandas
- Matplotlib
- Folium (for geospatial mapping)
- Google Colab
- GitHub

## Analysis Performed

### Crime Type Distribution
Analyzed the most frequent types of crimes reported in the dataset.

### Crime Trend by Year
Studied how crime counts changed across different years.

### Crime Distribution by Month
Analyzed seasonal crime trends and monthly variations.

### Crime by District
Identified districts with the highest number of reported crimes.

### Crime by Hour
Examined crime patterns throughout different hours of the day.

### Crime Hotspots
Used geographic coordinates to map crime incidents and generate heatmaps.


## Key Insights

- Theft and battery were among the most common crimes.
- Crime rates varied across districts, indicating geographic concentration.
- Certain hours of the day showed increased crime activity.
- Heatmaps revealed clear crime hotspots within the city.

The dataset contains reported crime incidents, including:

- Crime type
